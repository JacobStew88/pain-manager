# pain-manager
A webbased program, that will let users select a muscle group and 
pain-level to generate an exercise plan for them. 


Tech Stack

    Frontend: HTML, CSS, Handlebars, JavaScript
    Backend: Node.js, Express
    Database: MySQL with stored procedures
    Other Tools: Git, AJAX

Getting Started
1. Clone the repository

git clone https://github.com/your-username/wow-raid-tracker.git
cd wow-raid-tracker

2. Install dependencies

npm install

3. Set up database

    Create MySQL database
    Update db-connector.js with database credentials

4. Start the app
    node server.js

Visit http://localhost:3000 in your browser (port can be changed in app.js).
