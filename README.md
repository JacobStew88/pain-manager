# pain-manager
A webbased program, that will let users select a muscle group and 
pain-level to generate an exercise plan for them. 


Tech Stack

    Frontend: HTML, CSS, Handlebars, JavaScript
    Backend: Node.js, Express
    Database: MySQL with stored procedures
    Other Tools: Git, AJAX

Getting Startednpm 
1. Clone the repository

git clone 

2. Install dependencies

npm install

3. Start the app

    npm run development - runs the app in development mode to see real time edits.
    npm run production - runs server indefinintely until the next script is called npm run stop_production
    npm run stop_production - stops the server to have changes made to it.

Visit http://localhost:3000 in your browser (port can be changed in app.js).
